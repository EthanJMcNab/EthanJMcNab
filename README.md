## Projects

### GitHub Analytics Dashboard

A React dashboard for exploring public GitHub repositories through engineering health signals, repository activity, and AI-assisted audit findings.

- Integrates with the GitHub REST API to analyze repository metadata, language usage, issue health, readiness checks, and commit activity.
- Uses server-side API boundaries so GitHub and OpenAI keys are not exposed in browser code.
- Generates structured AI repository audits with evidence-based findings and recommended next actions.
- Includes CI, Dependabot, license, security policy, and contribution documentation.
